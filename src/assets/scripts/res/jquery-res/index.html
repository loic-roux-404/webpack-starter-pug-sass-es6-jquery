L'Arinella

Juillet 2018 - Aujourd'hui

Site web pour un proche, présentant une villa en Corse. Les options :

  • Un diaporama de la villa en accueil
  • Un descriptif des équipements de la villa propulsé par le plugin Jquery UI
  • Une galerie d'images dynamique pouvant être gérée dans la page d'administration
  • Une page d'administration protégée par un système de connexion. Elle permet de gérer des administrateurs (ajouter et supprimer) et la galerie grâce à MySQL et PHP (ajouter, modifier, réorganiser...).
  • Un système d'emailing (rubrique contact) basé sur PHPMailer et le SMTP de google.

Alt2crous

Mars 2018 - Avril 2018

Projet de site dynamique visant à trouver des restaurants autour des lieux d'études de Grenoble. Site web avec interface d'administration permettant de supprimer, ajouter et modifier des restaurants.
Ensuite les restaurants sont affichés en photo avec une description. En sélectionnant le lieu d'étude proche on trouve grâce à un algorithme les restaurants proches avec également un critère de prix.

Novembre 2018 - Janvier 2019

Projet de développement web sur le CMS Wordpress.
Selon le cahier des charges d'une autre équipe nous avons dû créer un thème Wordpress adapté à leurs demandes. Les options que nous avons pu ajouter sont :

  • Custom type, taxonomies et metaboxes
  • Pages d'archives selon les taxonomies
  • Personnalisation avec Redux framework : changement des polices sur les titres, choix des couleurs de fonds, choix du du diaporama et ses options sur la page d'accueil

  • Création de modules pour afficher des éléments spécifiques
